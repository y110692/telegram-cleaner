# Telegram Cleaner

Browser-first инструмент для локального разбора Telegram export.

## Возможности

- выбор папки с экспортом Telegram-чата целиком
- локальный показ текста, картинок, аудио, видео и файлов
- хранение прогресса и тегов в браузере
- экспорт заметок в markdown с вложениями
- скачивание результата одним `.zip` на Chrome и Safari
- запись экспорта прямо в папку в Chromium-браузерах

## Запуск

Из корня репозитория:

```powershell
.\start_fav_tinder.ps1
```

Или напрямую:

```powershell
python .\telegram-cleaner\server.py
```

После запуска откройте `http://127.0.0.1:8421`.

## Приватность

- сервер нужен только для раздачи статики локально или на домене
- экспорт Telegram выбирается пользователем в браузере
- `result.json` и медиа не отправляются на сервер
- в `index.html` задан CSP, запрещающий исходящие сетевые запросы из приложения
